# 正修及周邊環境避難應變資訊網 - 使用說明

## 網站概述

本網站為正修科技大學設計的避難應變資訊中心，提供校園師生與訪客完整的防災資訊、應變措施與安全指南。

## 網站內容

### 主要模塊

1. **避難空間資訊**
   - 校內避難集合點（勞工公園、體育館、禮堂等）
   - 周邊避難場所（勞工防災公園、鳥松區公所）
   - 容納人數與設施說明

2. **現行避難措施**
   - 硬體措施：緊急照明系統、應急設施、廣播系統
   - 軟體措施：定期演練、人員培訓、聯絡系統

3. **改善方向**
   - 短期應變：加強照明、道路巡檢
   - 中期改善：跨校合作、更新避難圖
   - 長期改善：排水改善、備援集合點、GIS 地圖更新

4. **應變指南**
   - 四步驟應對流程（保持冷靜、迅速疏散、到達安全地點、配合指揮）
   - 受影響人員類別說明

5. **資源下載**
   - 校園安全專題
   - 避難收容處所名冊
   - 防災演練計畫書

## 發佈流程

1. 在 Manus 管理界面打開項目
2. 點擊右上角「Publish」按鈕
3. 系統會自動生成公開網址
4. 可選：在「Domains」設定中綁定自訂網域
